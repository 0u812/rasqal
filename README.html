Overview

Rasqal is a free software/Open Source C library that handles RDF query syntaxes, query construction and query execution returning result bindings.

Rasqal was designed to work closely with the Redland RDF library but is entirely separate. It is intended to be a portable library working across many POSIX systems.

This is an beta quality library - the code and APIs are new however likely to change. See the todo list and the FAQS for general information. A summary of the changes can be found in the NEWS file, detailed API changes in the release notes and file-by-file changes in the CVS ChangeLog.

Rasqal provides:

  • A query construction API
  • A query execution engine
  • Query language support for RDQL
  • No memory leaks.
  • roqet standalone RDF query utility program

Supported Query Languages

RDF Data Query Language (RDQL)

Rasqal provides a syntax parser and query constructor for the RDQL language, as defined in RDQL - A Query Language for RDF, W3C Member Submission 9 January 2004 based on the earlier versions in Jena defined in RDQL RDF Data Query Language and the RDQL Grammar. The Jena RDQL Tutorial gives an introduction on the language. Rasqal currently passes the majority of the RDQL test suite tests. The tests not passed and further explanation are given in the todo list.
