Overview

Rasqal is a free software/Open Source C library that handles RDF query syntaxes and query construction..

Rasqal was designed to work closely with the Redland RDF library but is entirely separate. It is intended to be a portable library working across many POSIX systems.

This is an alpha quality library - the code and APIs are new and likely to change. See the to do list for more information. Changes can be found in the NEWS file or more detailed changes in the ChangeLog.
